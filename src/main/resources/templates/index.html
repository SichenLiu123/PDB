DNA-transcription factor binding sites This is a web server for analyzing the binding sites between double-standed DNA and transcription factors. It is able to allow users to search and analyze the DNA-transcription factor binding sites from a user-provided PDB ID(the structure identifier used in the Protein data Bank) or a PDB-formatted file. This web will help researchers to understand the molecular details of how the transcription factors recognize DNA and provide important clues for therapeutic targeting of oncogenic transcription factors.
